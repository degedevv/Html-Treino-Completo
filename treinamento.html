TREINAMENTO HTML

Projeto desenvolvido para prática e consolidação dos fundamentos de HTML.
  • Títulos (H1 - H6) / Parágrafos / Listas
  • Links / Imagem / Formatação de Texto / Quebra de Linha
  • Formulário

OBJETIVO

Este projeto tem como finalidade exercitar os principais elementos utilizados na construção de páginas web utilizando HTML puro. O foco está na estruturação correta do conteúdo e na organização semântica da página.

CONTEÚDOS PRATICADOS:

  • Títulos (h1 até h6)
  • Parágrafos
  • Listas ordenadas e não ordenadas
  • Links
  • Imagens
  • Formatação de texto
  • Quebra de linha
  • Linhas horizontais
